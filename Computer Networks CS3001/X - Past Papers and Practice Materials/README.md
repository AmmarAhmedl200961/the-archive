# Practice Materials for Computer Networks (CN) Course

Welcome to the **Practice Materials** folder for the Computer Networks course! This collection includes a variety of tutorials, practice questions, and interactive exercises sourced from online resources and the course textbook. These materials were used to reinforce key concepts like subnetting, network addressing, and more.

I've organized the resources below for easy access. Feel free to explore and practice!

## Subnetting Practice

## Textbook Exercises

- Practice the end-of-chapter exercises from the book (e.g., *Computer Networking: A Top-Down Approach* by Kurose & Ross).
- Verify your solutions using the provided answer keys.
- **Tip**: Focus on chapters related to network layer (e.g., IP addressing, subnetting) for the most relevant practice.

## Interactive Exercises

For hands-on learning, dive into the official interactive tools on the book's companion website:
- [Interactive Exercises](https://gaia.cs.umass.edu/kurose_ross/interactive/)

These applets cover topics like error detection, reliable data transfer, and routing algorithms. They're great for visualizing network protocols!
