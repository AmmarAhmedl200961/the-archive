## âš ï¸ Large Files & Git LFS

> **Note:** This repository uses **[Git Large File Storage (LFS)](https://git-lfs.github.com/)** to handle heavy course files like PDFs, datasets, videos, and other large resources. 

To properly clone this repository with all large files:

```bash
# Make sure you have Git LFS installed
git lfs install

# Clone the repository (LFS files will be pulled automatically)
git clone https://github. com/AmmarAhmedl200961/the-archive. git
```

If you've already cloned without LFS, run:
```bash
git lfs pull
```
