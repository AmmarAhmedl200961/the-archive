Guidelines for powerpoint presntations
1. first slide (topic and the name of presentor)
2. overview of your presentation
3. introductory paragraph
4. first body para
5. second body para
6.body para
7.conclusion
Note: Five to six Bullet points with phrases only for each slide.
Heading size = 36-38 (Times New Roman Bold)
Text size = 26-28 (Times New Roman)
Avoid bringing two sections of essay in one slide (i.e. introduction and first body para)
