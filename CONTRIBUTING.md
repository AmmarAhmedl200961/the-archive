## ðŸš€ How to Submit Changes

### Option 1: Pull Request (Preferred)

1. **Fork** the repository
2. **Create a branch** for your changes:
   ```bash
   git checkout -b add-course-materials
   ```
3. **Add your files** following the structure guidelines
4. **Commit** with a clear message:
   ```bash
   git commit -m "Add: Data Structures CS2001 - Quiz 3 solution"
   ```
5. **Push** to your fork and open a **Pull Request**

### Option 2: Open an Issue

If you're not comfortable with Git:
1. **Open an issue** describing what you'd like to add
2. **Attach files** or provide links to materials
3. We'll help integrate them into the archive!

### Option 3: Discussion

For suggestions or questions:
- Start a **[Discussion](https://github.com/AmmarAhmedl200961/the-archive/discussions)** to share ideas
